Módulo: Funciones Básicas

Bloques de código fundamentales diseñados para ejecutar tareas específicas y repetitivas.

1. Declaración de Función (Function Declaration)

La sintaxis clásica. Cuentan con "hoisting", lo que permite llamarlas antes de que sean declaradas en el código.

function inicializar(sistema) {
    return "Inicializando " + sistema + "...";
}
console.log(inicializar("Core"));

2. Expresión de Función (Function Expression)

La función se asigna a una variable. No tienen hoisting; la variable debe estar definida antes de su ejecución.

const analizar = function(datos) {
    return "Analizando " + datos.length + " bytes.";
};
console.log(analizar([1, 0, 1, 1]));

3. Funciones Flecha (Arrow Functions)

Sintaxis compacta introducida en ES6. No tienen su propio contexto this (heredan el del ámbito superior).

const encriptar = (cadena) => {
    return btoa(cadena);
};

// Si es una sola línea, el return está implícito:
const desencriptar = cadena => atob(cadena);

Módulo: Funciones Avanzadas

Estructuras complejas para manejar asincronía y encapsulamiento de variables.

1. Callbacks

Funciones que se pasan como argumentos a otras funciones para ejecutarse secuencialmente una vez que termina una tarea.

function extraerDatos(callback) {
    setTimeout(() => {
        console.log("Extracción completada.");
        callback();
    }, 1000);
}
extraerDatos(() => console.log("Procesando paquete..."));

2. Promesas (Promises)

Objetos que representan la finalización (o falla) eventual de una operación asíncrona, previniendo el "Callback Hell".

const saltarFirewall = new Promise((resolve, reject) => {
    let exito = true;
    if(exito) resolve("Brecha confirmada.");
    else reject("Acceso denegado.");
});

saltarFirewall
    .then(res => console.log(res))
    .catch(err => console.error(err));

3. Async / Await

Azúcar sintáctico sobre las promesas. Permite escribir código asíncrono que se lee como si fuera síncrono, facilitando el control de flujo.

async function iniciarInfiltracion() {
    try {
        console.log("Buscando vulnerabilidades...");
        const resultado = await saltarFirewall;
        console.log(resultado);
    } catch (error) {
        console.error(error);
    }
}

4. Closures (Clausuras)

Una función que recuerda el entorno (variables locales) en el que fue creada, incluso si se ejecuta fuera de ese entorno.

function generarToken(prefijo) {
    let sec = 0;
    return function() {
        sec++;
        return prefijo + "-" + sec;
    }
}
const tokenAcceso = generarToken("ROOT");
console.log(tokenAcceso()); // ROOT-1
console.log(tokenAcceso()); // ROOT-2
